# kataReact
React with express (TDD environment)

### GLOBAL DEPEDENCIES
 - NodeJs

### DEPEDENCIES
 - Install [yarn](https://yarnpkg.com/en/) and execute:

  ```
  $ yarn install
  $ sudo npm install -g nodemon
  $ sudo npm install -g karma-cli
  ```

### Run app:
  `$ yarn start`

### Run tests and linter:
  `$ yarn test`
